# Lofi Streams - Status

Live status page for all lofi streams.

**Live site:** https://ldraney.github.io/lofi-stream-frontend/

## Features

- Real-time stream status (updates every 60 seconds)
- Links to all platforms (YouTube, Twitch, Kick, Rumble)
- Links to all vibe sites
- Dark theme, mobile responsive

## Streams

| Stream | Platform | Vibe |
|--------|----------|------|
| Night City | YouTube | Cyberpunk/neon |
| Coffee Shop | Twitch | Cozy/warm |
| Arcade | Kick | Retro gaming |
| Library | Rumble | Quiet/studious |
